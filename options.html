About

This plugin will scrape the current page that you're visiting and look for valid RSS and Atom links, displaying any in a list if found. Bespoke behaviour is supported for pages that have an easily discernible feed, such as YouTube channels.

Further developments, options, and behaviours will appear down the road, so keep updated!
